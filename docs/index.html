An app which helps you perform breathing exercise based on Wim Hof Method (WHM). Breathing tempo and other variables can be tuned to your preferences. Some features are: + Breathing pace - how fast will you breath in first phase of the exercise. + Breathing animation - an animation to help synchronize breaths to the app's counter. + Recovery time - time in the last phase for your recovery after retention. + Number of rounds - the number of round to perform in breathing exercise. + Enabling / disabling some feature - some of the features like animation and phases tips can be disabled. + Tap to skip - you can double tap on the screen to skip remaining breaths/recovery time and move to the next phase. + Different Themes - Light | Dark | Bluish | Greenish | Reddish. + App Language - English | Polish.
